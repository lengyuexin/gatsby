### 左移

<P style={{background:'#FDB99B'}}>

```js
相当于 left*(2^right) 
10<<1  =>20 
```
</P>



### 右移


<P style={{background:'#FDB99B'}}>

```js
相当于 left/(2^right) 
100>>1  =>50
```
</P>



### 按位与


<P style={{background:'#FDB99B'}}>

```js
每一位都为1,结果才为1 
8 & 7  => 0  
1000 & 0111 => 0000 => 0
```
</P>



### 按位或


<P style={{background:'#FDB99B'}}>

```js
其中一位为 1,结果就是 1 
8 | 7 => 15 
1000 | 0111=> 1111=> 15
```
</P>




### 按位异或


<P style={{background:'#FDB99B'}}>

```js
每一位都不同,结果才为 1 
8 ^ 7 =>15 
1000 ^ 0111 => 1111 => 15
8 ^ 8  => 0
```
</P>
