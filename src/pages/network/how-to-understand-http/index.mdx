<Panel>
   全称：超文本传输协议。可以细分成超文本，传输，协议三个部分去理解：
</Panel>


1. 超文本与普通文本相对，除了纯文字外，还有**图片，音频，视频**等非纯文本形式的资源。
   最主要的是**超链接**，超链接可以跨页面，从而构成复杂的非线性的，网状的结构。

2. 传输需要有**请求方和应答方**，通常是客户端和服务端,也可以是服务端和服务端。
   **通信必须有服务端参与，浏览器只能做客户端**，两个浏览器是无法完成通信的。
3. 协议可理解为计算机世界里专门用来在两点之间传输数据的**约定和规范**，**并非实体**。
