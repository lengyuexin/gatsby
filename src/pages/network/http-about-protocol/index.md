### TCP/IP

- 现在常提到的TCP/IP往往不是单独的协议，而是**协议族**，其中最主要的两个协议是TCP和IP，常用的还有 UDP。
- 该协议族在网络分层协议栈中处于一个十分关键的位置，**上承应用层，下启链路层**。
- 其中提供数据通信的TCP和UDP协议位于**传输层**，解决寻址和路由问题的IP协议位于**网际层**。


### DNS

- 在 TCP/IP 协议中使用 IP 地址来标识计算机，但IP并不方便记忆，域名系统DNS应运而生。
- DNS在域名与IP之间建立了一种**映射关系**，既方便记忆，又能精确找到目标计算机地址。


### HTTPS

- HTTPS并不是一个全新的协议,而是运行在 **SSL/TLS** 协议上的 HTTP。
- SSL/TLS协议是一个负责**加密通信的可靠的安全协议**，建立在TCP/IP 之上。
