### TCP/IP

<P style={{background:'#FDB99B'}}>
现在常提到的TCP/IP往往不是单独的协议，而是协议族，其中最主要的两个协议是TCP和IP，常用的还有 UDP。<br/>
该协议族在网络分层协议栈中处于一个十分关键的位置，上承应用层，下启链路层。<br/>
其中提供数据通信的TCP和UDP协议位于传输层，解决寻址和路由问题的IP协议位于网际层。
</P>


### DNS


<P style={{background:'lightblue'}}>
在 TCP/IP 协议中使用 IP 地址来标识计算机，但IP并不方便记忆，域名系统DNS应运而生。<br/>
DNS在域名与IP之间建立了一种映射关系，既方便记忆，又能精确找到目标计算机地址。
</P>




### HTTPS

<P style={{background:'pink'}}>
HTTPS并不是一个全新的协议,而是运行在 SSL/TLS 协议上的 HTTP。<br/>
SSL/TLS协议是一个负责加密通信的可靠的安全协议，建立在TCP/IP 之上。
</P>
